# 爱关怀消息拉取接口


## 消息拉取接口（使用消息拉取的签名算法） 

***POST*** /fe/private/


输入：POST参数

请见上面表格公用参数。

参数名称 | 参数类型 | 是否必填 | 含义 | 固定值
--------|--------|---------|-----|-------
name | string  | 是 | 接口名称 | b.getPlanResultMessageList
version |  string  | 是 | 接口版本号 |
corp_id |  int  | 是 | 公司id 不可以为空 |
last_id | int  | 是 | 上一次最后一个id |
limit | int | 是  | 每页多少条 |
